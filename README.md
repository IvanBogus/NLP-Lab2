# Лабораторна робота 2. NLP-конвеєр

Проєкт для виконання лабораторної роботи 2 з аналізу та обробки природної мови(NLP).

Вхідний текстовий простір отримано на основі результатів ЛР1: `data/raw/lab1_news_titles.csv`.

Файл `reports/REPORT.md` генерується автоматично скриптом `scripts/run_pipeline.py`. Він потрібен як окремий звіт із актуальними результатами останнього запуску: обсягом вхідного тексту, кількістю токенів, TOP-10 слів, TOP-10 лем і порівнянням альтернативних методів.

## Структура

- `data/raw` - вхідні дані з ЛР1.
- `data/processed` - проміжні результати обробки: фільтрований текст, токени, леми, основи слів.
- `reports/REPORT.md` - автоматично згенерований звіт із поточними результатами.
- `reports/figures` - згенеровані візуалізації.
- `reports/metrics` - метрики, таблиці CSV і файли JSON.
- `scripts` - скрипти запуску.
- `src` - основний код NLP-конвеєра.

## Встановлення залежностей

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
```

## Запуск

```powershell
.\.venv\Scripts\python.exe scripts\run_pipeline.py
```

Після запуску оновлюються файли в `data/processed`, `reports/metrics`, `reports/figures` і `reports/REPORT.md`.

## Використані методи

У роботі реалізовано:

- фільтрацію вхідних даних;
- нормалізацію тексту;
- кілька типів токенізації;
- видалення стоп-слів;
- лематизацію;
- стемінг;
- визначення TOP-10 найчастіших слів;
- порівняння альтернативних підходів.

Основний код має fallback-варіанти на стандартній бібліотеці Python, але повне порівняння використовує `nltk`, `spacy`, `uk_core_news_sm` і `pymorphy3`.
